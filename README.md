# DA-pisos app

1) Primero descargue el código usando "git clone".
2) Luego a cmd (nodo, etc.):

   cd DA-pisos  --"file name"
   
   code .

4) Abra Visual Studio y escríbalos en Terminal:
   
   npm install
   
   npm start
